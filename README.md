# To Run the Project:

- signup on Auth0 and create a project. Refer (this link)[https://auth0.com/docs] for documentation.

- update AUTH ID and AUTH DOMAIN on client/src/config.ts and backend/src/authconfig.ts

- build serverless 
		```
			cd backend
			npm install
			sls deploy -v
		```

- build client

		```
			cd client
			npm install
			npm run start
		```
